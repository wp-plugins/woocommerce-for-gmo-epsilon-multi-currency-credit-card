== Installation ==

= In Japanese: =
= 必要条件 =

* WordPress 3.8 or greater
* WooCommerce 2.1 or greater
* PHP version 5.2.4 or greater
* MySQL version 5.0 or greater

= 自動インストール =
WordPressに標準装備のプラグインの新規追加からインストールしてください。<br />
プラグイン>新規追加<br />
検索窓にて「WooCommerce for GMO Epsilon Multi-currency Credit Card」と入力してクリックして頂けたら出てくると思います。<br />
そして、このプラグインが出てきたら「いますぐインストール」をクリックしたらインストールされます。

= 手動インストール =
手動でダウンロードファイルをダウンロードして頂いて、FTPソフト等でサーバーにアップして設定してください。

= In English: =
= Minimum Requirements =

* WordPress 3.8 or greater
* WooCommerce 2.1 or greater
* PHP version 5.2.4 or greater
* MySQL version 5.0 or greater

= Automatic installation =

Automatic installation is the easiest option as WordPress handles the file transfers itself and you don’t need to leave your web browser. To do an automatic install of WooCommerce for GMO Epsilon Multi-currency Credit Card, log in to your WordPress dashboard, navigate to the Plugins menu and click Add New.

In the search field type “WooCommerce for GMO Epsilon Multi-currency Credit Card” and click Search Plugins. Once you’ve found our eCommerce plugin you can view details about it such as the the point release, rating and description. Most importantly of course, you can install it by simply clicking “Install Now”.

= Manual installation =
The manual installation method involves downloading our plugin and uploading it to your webserver via your favourite FTP application.
